Создание простейших Web-страниц. Форматирование текста средствами языка HTML

    SRC=”URL” — адрес, указывающий браузеру, где находится рисунок.
    Рисунок должен храниться в графическом формате, поддерживаемом браузером. Например, для внедрения рисунка cat.jpg тег может иметь вид:
    IMG src="http://www.msfbntu.com/images/inf/cat.jpg" ALT="text" задает текст, который будет отображен браузером, не поддерживающим
    отображение графики или с отключенной подкачкой изображений. Обычно, это короткое описание
    изображения, которое пользователь мог бы увидеть на экране. Если данный атрибут отсутствует,
    то на месте рисунка большинство браузеров выводит пиктограмму, активизировав которую, пользователь может увидеть изображение.
